Manchester United 2021 Counterfactual Project Explained

A full briefing on the OGS project: what it asks, how it works, what the theory means, what the results say, and how to answer tough interviewer questions without overselling the model.

This project asks a sports-strategy question with a business lens: if Manchester United had not signed Cristiano Ronaldo in summer 2021, and had instead signed a control-oriented defensive midfielder, how might the season have looked up to Ole Gunnar Solskjaer’s exit and across the full 2021/22 campaign?

2021/22 Actual Points 58

Casemiro Scenario 64.47

What The Project Is Really About

On the surface, this is a football analytics project. Underneath, it is a decision-quality project about resource allocation, organizational diagnosis, and the trade-off between visible solutions and structurally useful ones.

The main strategic claim is simple: United may have solved the wrong problem in 2021. Ronaldo addressed star power and finishing. The team may have needed control, midfield protection, and defensive stability more urgently.

Core thesis: the project does not argue that Ronaldo alone caused the decline. It argues that United’s transfer decision may have been misaligned with the team’s deeper structural weakness.

Why This Question Matters

This is interesting because the Ronaldo transfer is famous, emotional, and easy to narrate lazily. That makes it a great case study for disciplined analysis. Rather than asking, “Was Ronaldo good or bad?”, the project asks a better question: “Did United allocate an elite resource to the capability they most needed?”

What Data The Project Uses

The project uses real public match-level Premier League data for three windows:

  • 2020/21 full season as the baseline
  • 2021/22 up to Ole Gunnar Solskjaer’s exit
  • 2021/22 full season

Main variables include date, opponent, venue, goals_for, goals_against, xG, xGA, shots_for, shots_against, result, and points.

Why Match-Level Data?

Because it is practical, interpretable, and honest. It lets us study how the team profile changed without pretending we have player-tracking or event-level tactical certainty. The project deliberately chooses clarity over fake sophistication.

How The Project Works End To End

  • Raw match CSV files are loaded from data/raw/.
  • The data is cleaned and standardized into a consistent schema.
  • Features are engineered, such as rolling xG difference, cumulative points, and expected-points proxies.
  • The project compares the 2020/21 baseline against the 2021/22 decline.
  • A counterfactual team profile is created where Ronaldo is replaced by a control-oriented midfielder.
  • A Poisson-based goal simulation generates match and season outcome distributions.
  • Charts, tables, and article-ready outputs are exported.

What The Descriptive Analysis Shows

Before modeling anything, the project checks whether United really got worse. The answer is yes.

  • 2020/21: 74 points, 1.95 points per match, positive xG difference
  • 2021/22 to Ole exit: 17 points, 1.42 points per match, negative xG difference
  • 2021/22 full season: 58 points, 1.53 points per match, almost flat xG difference

This matters because a good counterfactual should respond to a real observed problem, not invent one.

What “Control” Means In This Project

“Control” is not directly observed. It is proxied through team-level indicators such as lower xGA, fewer shots conceded, improved xG difference, and a more stable expected-points profile. That means the project does not claim to measure tactical control perfectly. It uses transparent, reasonable proxies for it.

What The Counterfactual Changes

The counterfactual assumes that without Ronaldo, United lose a small amount of attacking edge but gain defensive structure from a midfielder who improves control and protection.

Main Generic CDM Assumptions

  • attack_xg_multiplier = 0.94
  • control_xg_bonus = 0.05
  • defensive_xga_multiplier = 0.86
  • defensive_shots_against_multiplier = 0.90

Casemiro Scenario Assumptions

  • casemiro_defensive_xga_multiplier = 0.82
  • casemiro_control_xg_bonus = 0.07
  • Modeled as a stronger, named upper-bound case

The generic CDM is the main case because it is conceptually cleaner. Casemiro is secondary because he is a real player United later signed, which makes him useful but also more hindsight-sensitive.

For interview or article purposes, it helps to describe the generic CDM as a capability type that really existed in the market in summer 2021. Manuel Locatelli and Boubakary Soumare are useful reality anchors for that discussion, even though the simulation itself does not model one specific player.

The assumption values are best understood as scenario bands rather than discovered truths. The base case represents a strong first-team solution. The mild and strong cases then test whether the conclusion changes materially when that assumed midfield impact is softened or intensified.

The Core Theory Behind The Model

Counterfactual Analysis

A counterfactual asks what might have happened if one key choice had changed. It does not prove alternate history. It builds a disciplined scenario under explicit assumptions.

Causality vs Scenario Modeling

This project is not a causal identification study. It does not isolate true causal effect with experimental rigor. It is a scenario model designed to test whether a different squad-building logic produces meaningfully different outcomes.

Expected Goals

xG estimates the probability that a shot becomes a goal based on shot quality. It is useful because goals alone are noisy. xGA does the same for chances conceded.

Process vs Outcome

This is one of the most important ideas in the project. Outcomes are points and results. Process is xG, xGA, chance quality, shot profile, and expected points. A team can have decent short-term outcomes with a weak underlying process, or vice versa. The project tries to judge process, not just results.

Poisson Distribution

The simulation uses a Poisson model because football goals are low-frequency count events. A Poisson distribution is a simple and interpretable way to model counts over a fixed interval, which makes it a common baseline in football analytics.

Expected Points

Expected points is a model-based estimate of how many points a team “deserved” based on its underlying chance profile. It helps separate sustainable process from noisy outcomes.

Distribution, Not Certainty

The model does not return one magic answer. It returns a distribution of plausible season outcomes. That is why the project reports the mean, median, and percentile ranges rather than pretending the future is deterministic.

Main Results

To Ole’s Exit

  • Actual: 17.0
  • Generic CDM: 18.13
  • Casemiro-style: 19.21

Full Season

  • Actual: 58.0
  • Generic CDM: 61.16
  • Casemiro-style: 64.47

The key interpretation is that a better midfield fit likely helps, but does not transform the season completely. That supports a nuanced conclusion: United probably misdiagnosed the problem, but the club’s issues were broader than one transfer.

Robustness Check

To make sure the conclusion was not hanging on one convenient parameter set, the project also runs a mild, base, and strong generic-CDM sensitivity analysis.

  • Mild: 58.45 points
  • Base: 61.16 points
  • Strong: 64.56 points

That means the exact uplift is assumption-sensitive, but the directional conclusion survives: a more control-oriented midfield profile consistently looks more useful than the observed squad build. That is a stronger position than simply saying “my preferred multiplier produced the answer I wanted.”

What The Project Does And Does Not Claim

Safe Claims

  • United’s underlying profile worsened from 2020/21 to 2021/22.
  • A control-oriented midfielder likely improves the team in the model.
  • The generic case suggests modest improvement.
  • The Casemiro case suggests a larger upside under stronger fit.

Claims To Avoid

  • Ronaldo caused the collapse.
  • Casemiro definitely would have delivered 64 or 65 points.
  • One CDM would have fixed the club.
  • The model proves alternate history.

Major Limitations

  • The model is team-level, not player-level.
  • It is not a causal proof.
  • It translates midfield impact through simplified team-profile changes.
  • It does not capture dressing-room dynamics, wages, injuries, or coaching quality well.
  • Poisson is useful, but still a simplification of real football scoring dynamics.
  • The Casemiro case is partly hindsight-driven and should stay secondary.
Best one-line limitation: this is a transparent counterfactual scenario model, not a causal proof of alternate history.

How To Answer Interviewer Questions

Why did you choose this project?

I wanted to study a famous football decision through a strategy lens. The Ronaldo transfer is emotionally obvious, but analytically interesting because it raises a resource-allocation question about whether United solved the capability they most needed to solve.

Why not just write an opinion piece?

Because I wanted to move from narrative to evidence. The project uses data to test whether the squad-building logic made sense instead of simply reacting to the outcome.

Why use xG instead of just goals?

Goals are noisy and heavily affected by finishing variance. xG helps evaluate the quality of the process that produced the results.

Why use a Poisson model?

Football goals are low-frequency count events, and Poisson models are a common, interpretable baseline for modeling counts over fixed intervals.

Why not use a more advanced machine learning model?

Interpretability mattered more than complexity. I wanted a model whose assumptions I could explain, defend, and stress-test clearly.

Why is the generic CDM scenario more important than Casemiro?

The generic CDM is the cleaner analytical case because it tests a capability gap rather than overfitting to one player. Casemiro is useful as an illustrative upper-bound scenario because United later signed him.

What is the biggest insight?

That the team’s main problem may not have been lack of star quality. It was likely lack of balance, midfield control, and defensive protection.

What surprised you most?

That the generic CDM scenario helps, but only moderately. That actually made the conclusion more credible, because it suggests the club’s issues were broader than one transfer decision.

If an interviewer says, “Isn’t this just football?”

The topic is football, but the underlying problem is classic strategy: how organizations diagnose root causes, allocate scarce resources, and choose between visible and structurally useful solutions.

How To Explain The Whole Project Quickly

30-Second Version

I built a Python sports analytics project testing whether Manchester United solved the wrong problem in 2021. Using real match-level data, I compared the team’s 2020/21 and 2021/22 profiles, then simulated a counterfactual where Ronaldo was replaced by a control-oriented defensive midfielder. The model suggests United likely improve, but only moderately, which supports the broader conclusion that the club was addressing symptoms more than structural weakness.

2-Minute Version

I framed the Ronaldo transfer as a strategy question rather than a football argument. United signed an elite attacker, but the underlying team profile suggested the bigger weakness was midfield control and defensive stability. I built a clean Python pipeline around real public match-level data, engineered rolling and summary metrics, and first established that the team had clearly deteriorated from 2020/21 to 2021/22. Then I built a transparent counterfactual scenario where attacking output falls slightly without Ronaldo, but defensive stability improves with a strong defensive midfielder. I simulated outcomes using a Poisson model and compared actual versus counterfactual points. The generic midfield scenario improved the full season from 58 to about 61 points, while a stronger Casemiro-style case reached about 64.5. My conclusion was not that Ronaldo caused the collapse, but that United may have invested in the most visible solution rather than the most structurally useful one.
